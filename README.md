# Pets website

This is a website of pet orphanage for pet database maintenance that gives the opportunity to choose a pet and take it home.


## Applied skills

* HTML5
* Javascript ES8
* React + Redux

## Running

First install all dependencies:

```
npm install
```

Next run the app:

```
npm start
```
